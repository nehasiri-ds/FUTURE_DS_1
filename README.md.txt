# Superstore Sales Analysis

## Project Overview
This project analyzes a retail dataset to understand sales trends, customer segments, product performance, and profitability.

The goal is to extract useful business insights that can help companies improve their sales strategies and increase profits.

## Dataset
The dataset used is the Superstore dataset containing retail transaction data.

Number of records: 9994  
Number of columns: 21  

Main fields include:
- Order Date
- Customer Name
- Region
- Category
- Sub-Category
- Sales
- Profit
- Discount

## Tools and Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Steps

1. Data Loading
2. Data Cleaning
3. Data Exploration
4. Sales Analysis
5. Customer Segment Analysis
6. Regional Sales Analysis
7. Product Performance Analysis
8. Visualization and Insights

## Key Analysis Performed

- Sales trend analysis over time
- Sales distribution by category
- Regional sales performance
- Customer segment contribution
- Top selling products
- Profit vs Discount relationship

## Key Insights

- Technology category generated the highest revenue.
- The West region contributed the most sales.
- The Consumer segment placed the highest number of orders.
- Higher discounts often resulted in lower profit margins.
- Some products generate high sales but low profits due to excessive discounts.

## Business Recommendations

- Reduce excessive discounts to improve profitability.
- Focus marketing strategies on high-performing categories.
- Promote top-selling products more aggressively.
- Improve sales strategies in lower-performing regions.

## Conclusion

This project demonstrates how data analysis can help businesses identify opportunities to increase revenue and improve profitability.
